# EV model example

Example of an `evmodel` object created with `evprof` for testing
purposes. It has been created using an Open source data set of EV
charging sessions provided by
[ACN](https://acnportal.readthedocs.io/en/latest/).

## Usage

``` r
california_ev_model
```

## Format

### `california_ev_model`

An `evmodel` object.

- metadata:

  Information about the characteristics of the model

- model:

  Gaussian Mixture Models for connection times and energy
