# EV charging sessions example

Example of an charging sessions data set ready to use by `evprof`
functions. It is the open source data set downloaded from the [ACN-Data
website](https://ev.caltech.edu/dataset), transformed according to the
standard names defined by `evprof`

## Usage

``` r
california_ev_sessions
```

## Format

### `california_ev_sessions`

A `tibble` object with standard variable names defined by `evprof`

## Source

<https://ev.caltech.edu/dataset>
